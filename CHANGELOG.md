# Changelog

All notable changes to this project are documented here.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-07-11
### Added
- Initial public release.
- iCloud text-file inbox for capturing ideas from any device.
- Claude Code authors one-line ideas into fully-specified backlog rows.
- Read-only, self-refreshing, offline dashboard generated from the spreadsheet.
- Dashboard: sort by date, filter by status / type / source, separate open-item counts.
- Source tracking (items you requested vs. items Claude Code added).
- Sprint-end triage workflow with now-vs-later prompting.
- Dependencies, Blocked status, completion dates, and Claude notes.
- Built-in BL-000 smoke test that verifies the loop without touching code.
- Printable Word guides: deployment tutorial, usage cheat sheet, type cheat sheet, moving guide.
